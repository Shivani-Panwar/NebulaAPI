Files included in the package:
1. ApiDetails.csv : File that contains API call data, API key and API value
2. myApiTest : File that contains log from the test run
3. api_log : SQL file containing the schema and some test values

The project uses maven to run. Edit configurations to edit command line to "mvn clean spring-boot:run"
To change the DB credentials modify the application.properties file. Default credentials are already provided.
